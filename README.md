## Project Setup

### Install

```bash
# better-sqlite3-multiple-ciphers 依赖python环境
# 初始安装请在Anaconda Prompt中执行以下命令安装依赖
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows （需要管理员权限）
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
